# Yuki's Dream Journal ˚⟡˖ ࣪

## Description ˚ 𝜗𝜚˚⋆｡☆

This is a website to log your dreams. You can sign in with
discord and your dreams journals will private. You can only
write dream journals if you log in.

## Motivation

This is a website I made for my friend Vivi to log her dreams.
She tells me about her dreams a lot so I wanted to make her a
website to log her dreams in the morning before she comes to school
so she doesn't forget her dreams. She has the most interesting dreams.

# How I made this project

I made this project with blood sweat and tears. I really
wanted to give up. I did not like learning how to use all
of these new libraries and syntax.

This is a [T3 Stack](https://create.t3.gg/) project bootstrapped with `create-t3-app`.

- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Drizzle](https://orm.drizzle.team)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
